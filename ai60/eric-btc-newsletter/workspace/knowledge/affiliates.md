## Placement Rules

- Maximum 2-3 affiliate links per edition. Quality over quantity.
- Links must fit the content naturally. If the topic doesn't relate to any affiliate, include zero links.
- Never stack multiple affiliate links in the same paragraph.
- Always include the FTC affiliate disclosure from `disclaimers.md` when any affiliate link appears.
- Rotate affiliates across editions. Don't include the same partner in consecutive newsletters.
- Bybit requires a geo-restriction note in every placement (non-US only).
