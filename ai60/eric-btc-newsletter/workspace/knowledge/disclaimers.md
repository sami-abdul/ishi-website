# Disclaimers

Standard disclaimer text for every newsletter edition. The agent inserts these automatically.

## Newsletter Footer

Include this footer at the bottom of every edition:

```
This newsletter is for informational and educational purposes only. Nothing in this email
constitutes financial advice, investment advice, or a recommendation to buy or sell any
asset. Always do your own research before making any financial decisions. Past performance
does not guarantee future results.
```

## Affiliate Disclosure

Include this disclosure inline, before the first affiliate link in any edition:

```
This newsletter contains affiliate links. If you purchase through these links, we may earn
a commission at no extra cost to you. We only recommend products and services we genuinely
believe in.
```

If an edition contains no affiliate links, omit this disclosure.

## Sponsored Content Disclosure

For sponsored editions (triggered via `/sponsored-post`), include at the top of the post:

```
This edition is sponsored by [SPONSOR NAME]. While this content is sponsored, all opinions
and analysis are our own.
```

## Placement Rules

| Disclosure | Where | When |
|------------|-------|------|
| Newsletter footer | Bottom of every edition | Always, no exceptions |
| Affiliate disclosure | Before first affiliate link | Any edition with affiliate links |
| Sponsored disclosure | Top of post, below title | Sponsored editions only |

## Compliance Notes

- FTC Endorsement Guides require clear and conspicuous disclosure of material connections
- Affiliate links must be identifiable (do not disguise or cloak affiliate URLs in a misleading way)
- Sponsored content must be labeled before the reader encounters any sponsored claims
- "Not financial advice" does not replace the need for accurate, sourced data in every edition
