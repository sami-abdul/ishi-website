# Community Plugins Documentation

## Overview
OpenClaw maintains a curated registry of community-developed plugins that meet specific quality standards. The platform accepts contributions through pull requests when submissions satisfy established criteria.

## Submission Requirements
To list a plugin, developers must ensure:

- The package is available on npmjs and installable via the `openclaw plugins install` command
- Source code resides in a public GitHub repository
- Documentation and issue tracking are included in the repository
- The project demonstrates active maintenance through recent updates or responsive issue management

## Contribution Process
Prospective contributors should submit a pull request containing the plugin name, npm package identifier, GitHub URL, brief description, and installation command.

## Quality Standards
The review process prioritizes plugins that are functional, well-documented, and operationally sound. "Low-effort wrappers, unclear ownership, or unmaintained packages may be declined."
