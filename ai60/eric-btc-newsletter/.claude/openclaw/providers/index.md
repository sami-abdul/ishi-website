# Model Providers Documentation Summary

OpenClaw supports numerous LLM providers through a straightforward setup process. Users must authenticate with their chosen provider and configure the default model using the format `provider/model`.

## Key Setup Steps

The documentation indicates that implementation requires two main steps: "Authenticate with the provider (usually via `openclaw onboard`)" and then designate a default model in the configuration file.

## Available Providers

The platform offers integration with 24+ providers, including major options like Anthropic, OpenAI, Mistral, and specialized services such as Ollama for local models and Deepgram for audio transcription. The complete list encompasses cloud-based services, local solutions, and privacy-focused alternatives.
