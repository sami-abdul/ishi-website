# OpenClaw Credits

## Project Name Origin
OpenClaw combines "CLAW" with "TARDIS," reflecting the whimsical concept of a space lobster requiring both temporal and spatial navigation capabilities.

- **Peter Steinberger** ([@steipete](https://x.com/steipete)) -- Project founder and lead developer
- **Mario Zechner** ([@badlogicc](https://x.com/badlogicgames)) -- Pi creator and security researcher
- **Clawd** -- The space lobster mascot who influenced the project's naming
