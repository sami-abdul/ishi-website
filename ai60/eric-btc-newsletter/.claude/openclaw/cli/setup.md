# OpenClaw Setup Documentation

## Overview
The `openclaw setup` command initializes the OpenClaw configuration file and workspace environment.

## Command Function
This command creates `~/.openclaw/openclaw.json` and establishes the agent workspace structure needed for OpenClaw operations.

## Basic Usage
```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

## Wizard Option
To launch the interactive onboarding wizard during setup:
```bash
openclaw setup --wizard
```
