# ClawBot Documentation

## Overview

The page documents `openclaw clawbot`, described as a "Legacy alias namespace kept for backwards compatibility."

## Supported Commands

The documentation indicates one currently supported alias:

- `openclaw clawbot qr` maintains the same functionality as the modern command `openclaw qr`

## Migration Guidance

Users are encouraged to transition to updated command syntax. The recommended migration path is:

`openclaw clawbot qr` → `openclaw qr`
