# OpenClaw Skills Documentation

## Overview

This page documents the `openclaw skills` command, which allows users to examine available skills and their requirements.

## Key Commands

The documentation lists four primary commands:

- `openclaw skills list` -- display all skills
- `openclaw skills list --eligible` -- show only qualifying skills
- `openclaw skills info <name>` -- retrieve details about a specific skill
- `openclaw skills check` -- validate skill status
