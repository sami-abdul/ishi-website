# Webhooks Documentation

## Overview

The `openclaw webhooks` command provides webhook helpers and integrations, specifically for Gmail Pub/Sub and general webhook functionality.

## Related Resources

- Webhook documentation: [Webhook](/automation/webhook)
- Gmail Pub/Sub integration: [Gmail Pub/Sub](/automation/gmail-pubsub)

## Gmail Setup and Usage

To configure and run Gmail webhooks:

```bash
openclaw webhooks gmail setup --account you@example.com
openclaw webhooks gmail run
```

For comprehensive details on Gmail Pub/Sub functionality, refer to the dedicated [Gmail Pub/Sub documentation](/automation/gmail-pubsub).
