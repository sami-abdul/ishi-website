# DNS Documentation

## Overview

This page documents the `openclaw dns` command, which provides DNS helper functionality for wide-area discovery using Tailscale and CoreDNS, with current support for macOS and Homebrew CoreDNS installations.

## Command Usage

```bash
openclaw dns setup
openclaw dns setup --apply
```
