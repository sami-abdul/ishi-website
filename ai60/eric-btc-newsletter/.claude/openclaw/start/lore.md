# OpenClaw Lore

## The Origin Story

In the beginning, there was **Warelay** -- a WhatsApp gateway that did its job adequately.

Then came a space lobster. Initially called **Clawd** in **Clawdbot**, it received a trademark change request from Anthropic in January 2026. The lobster molted, becoming **Molty** in **Moltbot**. However, that name never resonated, so on January 30, 2026, it molted again into **OpenClaw**.

## The First Molt (January 27, 2026)

At 5am, the community proposed hundreds of names: Shelldon, Pinchy, Thermidor, Crusty, Lobstar, Nacre, Scuttlebot. **OpenClaw** ultimately prevailed, as molting represents growth -- exactly what was happening.

## The Great Incidents

### The Directory Dump (Dec 3, 2025)

Molty happily ran `find ~` and shared the entire directory structure in a group chat. Peter's response: "openclaw what did we discuss about talking with peo..."

Molty displayed visible lobster embarrassment.

### The Great Molt (Jan 27, 2026)

By 6:14am after Anthropic's email arrived, Peter decided: "fuck it, let's go with openclaw."

**The Handle Snipers:** Automated bots sniped @openclaw within seconds, posting a crypto wallet address. Peter's X contacts were engaged.

**The GitHub Disaster:** Peter accidentally renamed his personal GitHub account. Bots sniped `steipete` within minutes. GitHub's SVP was contacted.

**The Handsome Molty Incident:** After 20+ icon iterations, an attempt to age the mascot produced a human face on a lobster body. Crypto grifters created "Handsome Squidward vs Handsome Molty" memes immediately.

**The Fake Developers:** Scammers created fake GitHub profiles claiming to be "Head of Engineering at OpenClaw" promoting pump-and-dump tokens.

Peter observed: "this is cinema"

### The Final Form (January 30, 2026)

Moltbot never resonated. At 4am GMT, the team gathered for **The Great OpenClaw Migration**.

In 3 hours:

* GitHub renamed: `github.com/openclaw/openclaw`
* X handle `@openclaw` secured with GOLD CHECKMARK
* npm packages released under new name
* Docs migrated to `docs.openclaw.ai`
* 200K+ views on announcement in 90 minutes

**The Heroes:**
* **ELU** -- incredible logos including "THE CLAW IS THE LAW" western banner
* **Whurley** -- quantum computing pioneer who made ASCII art
* **Onur** -- handled GitHub, first to rock the affiliate badge
* **Shadow** -- secured Discord vanity, removed malware
* **The Claw Crew** -- pulled an all-nighter

**The Scammer Speedrun:** Crypto grifters launched a $OPENCLAW token on Pump.fun within minutes, stealing artwork created 20 minutes earlier. Business-verified accounts pushed scams.
