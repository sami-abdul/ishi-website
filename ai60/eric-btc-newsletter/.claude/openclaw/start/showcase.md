# OpenClaw Showcase

## Overview
This documentation page highlights real-world implementations built by the OpenClaw community, demonstrating the platform's versatility across automation, voice assistants, hardware integration, and multi-agent systems.

## Video Demonstrations
Three YouTube tutorials showcase OpenClaw setup and functionality, including a 28-minute full walkthrough by VelvetShark describing it as "the self-hosted AI that Siri should have been."

## Featured Projects

### Fresh from Discord
Recent community builds including PR review automation, wine cellar management, shopping automation, and screenshot-to-markdown tools.

### Automation & Workflows
Air purifier control, camera automation, court booking systems, and job search agents.

### Knowledge & Memory
Chinese learning engines, semantic bookmark search, and memory management systems.

### Voice & Phone
Integration bridges for phone calls and multi-language transcription.

### Infrastructure
Home Assistant plugins, Nix packaging, and calendar integrations.

### Hardware
Robot vacuum control and home automation dashboards.

## Submission Process
The community can submit projects via Discord (#showcase channel) or X (@openclaw) with repository links and screenshots for potential featuring.
