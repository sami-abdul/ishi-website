# OpenClaw Documentation Directory

This page is a curated index. If you are new, start with [Getting Started](/start/getting-started).

For a complete overview, the site offers access to all linked pages via the "Docs hubs" section.

## Content Organization

The documentation is divided into five main sections:

**Getting Started** covers essential setup topics including configuration, assistant setup, and node pairing.

**Providers and UX** focuses on communication platforms like Telegram, Discord, and Mattermost, plus media handling.

**Companion Apps** lists platform-specific applications for macOS, iOS, Android, Windows, and Linux.

**Operations and Safety** addresses automation features (cron jobs, webhooks) and security considerations.

The page also references a complete documentation file available at an external URL for comprehensive discovery.
