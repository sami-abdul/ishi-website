# OpenClaw Documentation Index

This documentation hub provides comprehensive guidance for OpenClaw, organized across multiple categories.

## Key Starting Points

The docs recommend beginning with "Getting Started" for new users, with additional foundational resources including quick start guides, onboarding procedures, and setup instructions. A local gateway dashboard is available at http://127.0.0.1:18789/.

## Major Documentation Sections

**Installation & Infrastructure:** Docker and Nix installation methods, update procedures, and experimental Bun workflow support.

**Foundational Concepts:** Architecture overview, feature descriptions, agent runtime mechanics, memory systems, and streaming/chunking approaches. The documentation covers "multi-agent routing," session management, and model failover capabilities.

**Integration Channels:** Support for WhatsApp, Telegram, Slack, Discord, Signal, and other communication platforms, plus webhook and Gmail Pub/Sub integrations.

**Gateway Operations:** Comprehensive runbook covering network models, pairing, health monitoring, security, and troubleshooting procedures.

**Tools & Capabilities:** Documentation for tools including PDF processing, browser control, exec utilities, cron jobs, and voice features. References to "Thinking + verbose" modes and sub-agent functionality.

**Platform Support:** Detailed guides for macOS, iOS, Android, Windows (WSL2), and Linux deployments, with extensive macOS companion app documentation covering menu bar, voice wake, and XPC integration.

**Configuration & Templates:** Reference materials for AGENTS, BOOTSTRAP, HEARTBEAT, IDENTITY, SOUL, TOOLS, and USER templates.
