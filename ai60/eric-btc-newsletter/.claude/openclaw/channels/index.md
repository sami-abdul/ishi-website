# Chat Channels Documentation

OpenClaw integrates with numerous messaging platforms through its Gateway system. The platform supports text communication universally, though media and reaction capabilities vary by channel.

## Key Offerings

The system provides access to "any chat app you already use," encompassing 24+ channels including popular services like Discord, Telegram, WhatsApp, and Slack alongside enterprise solutions such as Microsoft Teams and self-hosted options like Nextcloud Talk.

## Notable Features

**Setup flexibility** is a primary advantage—users can "run simultaneously" multiple channels with automatic routing based on conversation context. The documentation notes that "Telegram" typically offers the quickest configuration, requiring only a bot token, whereas WhatsApp demands QR authentication.

**Safety mechanisms** include mandatory DM pairing and allowlist enforcement across channels. Group functionality adapts per-platform specifications.
