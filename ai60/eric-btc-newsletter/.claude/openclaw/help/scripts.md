# Scripts Documentation

The `scripts/` directory provides helper utilities for local development and operational tasks within the OpenClaw project.

## Key Points

**When to use scripts:** The documentation advises preferring CLI tools when available. For instance, auth monitoring should use `openclaw models status --check` rather than direct scripts. Scripts are considered optional unless explicitly mentioned in documentation or release checklists.

**Important consideration:** Scripts may be environment-specific, so reviewing their contents before execution on a new system is recommended.
